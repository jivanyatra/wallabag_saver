How to Add a New Client in Wallabag

  1. Log in to your Wallabag web interface as an administrator.
  2. Go to Settings → API clients management.
  3. Click Add a client.
  4. Fill in the form:
    • Name: Choose a name (e.g., "Wallabag Saver Extension").
    • Redirect URI: You can use urn:ietf:wg:oauth:2.0:oob or leave blank for password grant.
    • Leave other fields as default unless you know you need to change them.
  5. Click Save.
  6. Copy the Client ID and Client Secret from the list. You will need these for the extension.

For more details, see the Wallabag API documentation.
